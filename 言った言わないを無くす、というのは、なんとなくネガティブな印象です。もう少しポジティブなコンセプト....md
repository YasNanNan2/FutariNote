# **User Story Map: Futari Note (Final)**

Concept: 二人の毎日を「チーム」として楽しみ、未来と感謝を育むパートナーシップアプリ  
Core Persona: 互いの時間を尊重し、心地よい家庭を築きたい共働き夫婦  
Phase: Release 1 (MVP) & Release 2 Plan

## **1\. Activity: 導入・連携 (Onboarding)**

**Goal:** 二人がデジタル上でつながり、新しい「チーム生活」のスタートを切る。

| Narrative (User Task) | Release | User Story (Details) | Who | Why (Value) |
| :---- | :---- | :---- | :---- | :---- |
| **1-1. アカウントを作成する** | **MVP** | LPからニックネームとEmailだけで即座に登録する | ホスト | 期待感が高いうちに、スムーズに二人の空間を作るため |
|  | **MVP** | 自分のテーマカラーを設定する | ユーザー | 二人の個性を表現し、互いの存在を画面上で感じるため |
| **1-2. パートナーを招待する** | **MVP** | 専用の招待URLを発行し、LINE等で送付する | ホスト | 「これ一緒にやろう！」と気軽に声をかけ、巻き込むため |
|  | **MVP** | 招待コード自体も確認・コピーできるようにする | ホスト | 確実にパートナーを迎え入れるための予備手段として |
|  | **MVP** | パートナーの登録完了通知を受け取る | ホスト | チーム結成の瞬間をリアルタイムに共有し、喜び合うため |
| **1-3. リンクから参加する** | **MVP** | URLタップで招待コード入力済みの登録画面を開く | ゲスト | ストレスなく、歓迎された状態で参加するため |
|  | **MVP** | 「〇〇さんからの招待」という表示を確認する | ゲスト | パートナーとのつながりを実感し、安心して参加するため |
|  | **MVP** | 登録完了と同時に、ペアリング済みのホームへ移動する | ゲスト | すぐに二人の共同プロジェクト（生活）を始めるため |

## **2\. Activity: タスク管理 (Organize Tasks)**

**Goal:** 互いの負担と貢献を見える化し、阿吽の呼吸で回る心地よい連携を生む。

| Narrative (User Task) | Release | User Story (Details) | Who | Why (Value) |
| :---- | :---- | :---- | :---- | :---- |
| **2-1. タスクを登録する** | **MVP** | タスク名・日付・担当・カテゴリを入力して追加する | 起案者 | 「誰が・いつ・何を」を共有し、安心して任せ合うため |
|  | **Rel 2** | 定期的な家事（ゴミ出し等）の「繰り返し設定」をする | 家事担当 | ルーチンワークを自動化し、二人の時間を増やすため |
| **2-2. タスクをメンテナンスする** *(CRUD: Update/Delete)* | **MVP** | 登録済みのタスク内容（日付変更など）を修正する | ユーザー | 生活の変化に柔軟に合わせ、無理なく続けるため |
|  | **MVP** | 不要になったタスクを削除する | ユーザー | リストを常にクリアに保ち、気持ちよく使うため |
| **2-3. タスクを確認する** | **MVP** | 「未完了タスク」を一覧で確認する | ユーザー | 今日の二人のミッションを把握し、協力し合うため |
|  | **MVP** | 「自分の担当」だけに絞り込んで表示する | ユーザー | 自分の役割に集中し、確実にチームに貢献するため |
| **2-4. スケジュールを俯瞰する** | **MVP** | カレンダー上でタスク（点）と目標（旗）の分布を見る | ユーザー | 日々の忙しさと、楽しみな予定（目標）のバランスを見るため |
|  | **MVP** | カレンダーの日付タップで、その日の詳細を見る | ユーザー | その日をどう過ごすか、二人で見通しを立てるため |

## **3\. Activity: 目標共有 (Share Goals)**

**Goal:** 目の前の家事を、二人の「未来の夢」を叶えるためのステップに変える。

| Narrative (User Task) | Release | User Story (Details) | Who | Why (Value) |
| :---- | :---- | :---- | :---- | :---- |
| **3-1. 目標を共有する** | **MVP** | 共通の目標（タイトル・金額・期限）を設定する | 夫婦 | 二人が目指すワクワクする未来（ビジョン）を一致させるため |
|  | **MVP** | 目標を象徴するアイコンを設定する | 夫婦 | 夢を視覚的にイメージし、見るたびにモチベーションを上げるため |
| **3-2. 目標をメンテナンスする** *(CRUD: Update/Delete)* | **MVP** | 目標の金額や期限を修正する | 夫婦 | 二人のペースに合わせて、計画を最適化し続けるため |
|  | **MVP** | 達成・頓挫した目標を取り下げる（削除/アーカイブ） | 夫婦 | 次の新しい夢に向かってフォーカスするため |
| **3-3. 進捗を更新する** | **MVP** | 現在の貯金額や数値を更新し、進捗バーを見る | ユーザー | 夢に近づいている「前進感」を二人で味わうため |

## **4\. Activity: 実行・完了 (Do & Complete)**

**Goal:** タスク消化を「事務作業」から、互いの貢献を称える「達成イベント」にする。

| Narrative (User Task) | Release | User Story (Details) | Who | Why (Value) |
| :---- | :---- | :---- | :---- | :---- |
| **4-1. タスクを完了する** | **MVP** | タスクをチェックして完了状態にする | 実行者 | チームへの貢献を記録し、自己効力感を得るため |
|  | **MVP** | **完了時に紙吹雪のエフェクトを見る** | 実行者 | 小さな達成を祝福され、ポジティブな気持ちになるため |
|  | **MVP** | 間違えて完了した操作を取り消す（Undo） | 実行者 | 操作ミスを気にせず、リラックスして使うため |
| **4-2. 相手に知らせる** | **MVP** | パートナーの完了通知（バッジ等）を受け取る | パートナー | パートナーの活躍を知り、感謝のきっかけを得るため |

## **5\. Activity: 感謝・振り返り (Appreciate)**

**Goal:** アプリ体験のピーク。感謝と愛情を可視化し、二人の絆を深める。

| Narrative (User Task) | Release | User Story (Details) | Who | Why (Value) |
| :---- | :---- | :---- | :---- | :---- |
| **5-1. 感謝を伝える** | **MVP** | 履歴からワンタップで感謝スタンプを送る | パートナー | 「ありがとう」の気持ちを、カジュアルに絶え間なく届けるため |
|  | **MVP** | 感情に合ったスタンプ（Love, お疲れ様等）を選ぶ | パートナー | 言葉にしにくい労りや愛情を、スタンプに乗せて伝えるため |
|  | **Rel 2** | 送ったスタンプを取り消す | パートナー | （操作ミスへの対応） |
| **5-2. 貢献を振り返る** | **MVP** | タイムラインで二人の活動履歴を見る | 夫婦 | 二人で積み重ねた日々（ログ）を愛おしく振り返るため |
|  | **MVP** | **感謝の総数（Thanks Count）を見る** | 夫婦 | 交わした感謝の数が、二人の「信頼残高」として増える喜びを感じるため |

## **6\. Activity: 設定・管理 (Settings)**

**Goal:** 自分らしい使い方で、長く快適にチーム運営を続ける環境を整える。

| Narrative (User Task) | Release | User Story (Details) | Who | Why (Value) |
| :---- | :---- | :---- | :---- | :---- |
| **6-1. アカウント管理** | **MVP** | プロフィール（名前・カラー）を変更する | ユーザー | 生活の変化に合わせて、今の自分たちにフィットさせるため |
|  | **MVP** | アカウントを削除（退会）する | ユーザー | デジタルライフを自身の意思でコントロールするため |
| **6-2. アプリ設定** | **MVP** | 通知設定をON/OFFする | ユーザー | オフの時間も大切にしつつ、心地よい距離感でつながるため |
| **6-3. 連携管理** | **Rel 2** | パートナーとの連携を解除する | 夫婦 | （連携のリセットや解消への対応） |
